
#NNTrainer Visualiser

a vscode extension for visualizing NNTrainer architecture and weights.

How to install

1. Install Node.js
2. package the extensions 
npm install -g @vscode/vsce
cd tools/nntrainer-visualizer
vsce package --no-dependencies --allow-missing-repository
3. Install in vscode
code --install-extension nntrainer-visualizer-1.0.0.vsix
4. Reload the vscode
5. open any .ini file -> right click -> open in nntrainer visualizer -> click " Load .bin weights in the top bar





Quick Install (prebuilt .vsix)
1. download the .vsix file
2. Install into vscode or (ctrl+shift+X ) -> Install from VSIX pick the file 
3. reload Vscode
4. open any .ini file -> right click -> open in nntrainer visualizer -> click " Load .bin weights in the top bar
